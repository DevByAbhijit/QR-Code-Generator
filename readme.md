# 🔗 QR Code Generator
 
A simple, beginner-friendly web application that generates QR codes instantly from user input such as text, links, or messages. This project was built as part of my web development learning journey to explore how QR code generation works and to practice integrating external libraries/APIs into a JavaScript project.
 
---
 
## 📖 About the Project
 
This QR Code Generator is a lightweight, browser-based application built using **HTML, CSS, and JavaScript**. Users can type or paste any text or URL into an input field, and the app instantly generates a corresponding QR code that can be scanned by any QR code reader.
 
The goal of this project was to practice:
- Capturing and handling user input in real time
- Working with a third-party library/API to generate dynamic visual content
- Updating the DOM dynamically to display generated content
- Building a simple, functional, and responsive user interface
As a beginner project, this focuses on core front-end fundamentals and understanding how to integrate external tools into a JavaScript application.
 
---
 
## ✨ Features
 
- Generates a QR code instantly from any text or URL input
- Real-time QR code generation as you type or on button click
- Simple, clean, and easy-to-use interface

## 🚀 Getting Started
 
To run this project locally:
 
1. Clone the repository
```bash
   git clone https://github.com/your-username/qr-code-generator.git
```
2. Navigate to the project folder
```bash
   cd qr-code-generator
```
3. Open `index.html` in your browser
No installation, build tools, or dependencies are required — it runs directly in the browser.
 
---
 
## 📌 Usage
 
1. Open the app in your browser.
2. Type or paste the text, link, or message you want to convert.
3. Click the "Generate" button (or wait for real-time update).
4. Scan the generated QR code using any QR scanner app.
